Journey

My journey started when I was a little kid where I would always try to see if I could make things easier. For better or worse, this mindset stuck with me into university and my career! One of my earliest workplace projects included creating a comparison program between an internal and external excel report that contained thousands of rows of box-ID. With no prior knowledge, I used excel macros to locate the files and the function 'VLOOKUP' to check if there were any IDs missing in each report. The check would then reverse-check to double-check it's work. I identified many boxes which were either missing were destroyed despite being billed.
