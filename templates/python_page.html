Data Types

Data types include primitive types like integers (int), floating-point numbers (float), and complex numbers (complex) for numerical computations.
The bool type represents Boolean values, which can be either True or False.
Python also provides the str type for handling text, supporting Unicode characters for internationalization.
For collections, Python offers several built-in types: lists (list), tuples (tuple), and ranges (range) for ordered sequences; sets (set and frozenset) for unordered collections with no duplicate elements; and dictionaries (dict) for key-value pairs.

Operators

Arithmetic operators such as + (addition), - (subtraction), * (multiplication), / (division), // (floor division), % (modulus), and ** (exponentiation) handle mathematical operations.
Comparison operators like == (equal to), != (not equal to), >, <<, >=, and <= compare values and return Boolean results.
Logical operators and, or, and not are used to combine or invert Boolean expressions.
Bitwise operators, including & (and), | (or), ^ (xor), ~ (not), << (left shift), and >> (right shift), perform operations at the bit level.
Assignment operators such as =, +=, -=, *=, and /= help in assigning and updating values. Additionally, identity operators is and is not to check if two variables reference the same object, and membership operators in and not in to check if a value exists within a sequence or collection.

Loops

Using the Operator 'for', data types, and conditional logic, python is able to iterate and repeat a set of instructions which is very powerful when repetition is required in a project. For example, if we are required to print out numbers up to 5, we could simply print out 1, 2, 3, 4, 5 individually. However, printing up to 100 or 1000 numbers will take too much time so we can create a 'for' loop, to print the numbers for us, as long as the condition (x<=100) is met:


for x in range(100):
print(x)

Functions

Defined using the def keyword, functions encapsulate a set of instructions that perform a specific task and can be executed whenever needed. Functions can accept parameters to allow input variability and return values to provide output results. Python supports default parameter values, keyword arguments, and variable-length argument lists, offering great flexibility in function definitions

Dictionaries and Lists (with comprehension)

Python dictionaries and lists are versatile data structures that provide dynamic ways to store and manipulate collections of items. A list is an ordered collection that allows for indexing, slicing, and iteration, making it ideal for storing sequences of items. Lists can contain elements of different data types, including other lists, and support a variety of operations like appending, inserting, removing, and sorting elements. Dictionaries, on the other hand, are unordered collections of key-value pairs, where each key is unique. They allow for fast lookups, additions, and deletions of values based on their keys. Dictionaries are particularly useful for representing structured data, such as objects and records, and for implementing associative arrays or hash maps. Both data structures are mutable, enabling in-place modifications.

Object-oriented Programming

Python's object-oriented programming (OOP) paradigm allows for the creation of classes and objects, fostering code organization, reusability, and abstraction. Classes serve as blueprints for objects, encapsulating data in attributes and behavior in methods. In Python, a class is defined using the class keyword, and objects are instantiated from classes to represent specific instances. OOP principles such as inheritance, encapsulation, and polymorphism are well-supported in Python. Inheritance allows new classes to derive from existing ones, promoting code reuse and hierarchical relationships. Encapsulation hides the internal state of objects, exposing only necessary components through methods and properties, thereby enhancing data integrity and security.
